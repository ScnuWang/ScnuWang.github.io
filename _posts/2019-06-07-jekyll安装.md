Jekyll安装步骤

### 安装（Windows）

1.  安装**Ruby+Devkit**环境，下载地址：https://rubyinstaller.org/downloads/
2.  要勾选执行`ridk install`命令，和加入环境变量选项
3. 安装jekyll：`gem install jekyll bundler`
4.  测试：`jekyll -v`

### 创建博客

确保jekyll已经安装好之后，

1. 新建一个博客，取名为kblog,名字自定义

   ```
   jekyll new kblog
   ```

2. 本地运行刚刚创建的博客

   ```
   cd kblog
   bundle exec jekyll serve
   ```
